Here’s a list of everything I’ve referenced while building this series. Huge thanks to all the pros for their work. It might be missing a few things since I didn’t originally plan on writing these articles. My apologies.

Meanwhile, I’ll also be removing the “References” section from individual articles.

Articles

Section titled “Articles”
  • VRChat Official Documentation: VRChat

Found the official docs later. It’s all in English; I mostly read through it using machine translation.

  • Blender Official Documentation: Blender 2.93 Reference Manual

As we all know, official docs are always impossible to understand.

  • Unity User Manual (2019.4 LTS)

  • VRChat Model Creation and Upload Overview (201912) - Xingkong Yue Ling -博客园

The first article I read. It was the top result when Googling vrchat upload mmd model. Very comprehensive, but the tutorial is a bit dated.

  • Unity Animator State Machine - 1024Sou

Covers various Unity animation features (though I didn’t understand the last part).

  • The Role of Write Defaults - CSDN Blog

Reference used for the VRLabs Marker section.

  • Blender - 2.8x - Screencast Keys (v3.0) - Plugin Introduction and Installation

Plugin reference for displaying keypresses in the Blender head-swapping video.

  • 【Notes】Dynamic Bone (& Jiggle) Research Notes

I didn’t understand it, but I was deeply impressed. Three articles on adding Dynamic Bone collisions.

  • Detailed Explanation of Unity Animation System 3: How to Play and Switch Animations? - Zhihu

Finally understood the parts I didn’t get before!

  • Unity Cloth Simulation Tutorial

  • Introduction to Simple and Common Unity Particle System Functions - Tencent Games Institute

  • Installing a Specific Version of Unity3D on Linux - CSDN Blog

  • How to Delete and Merge Overlapping/Adjacent Points in Blender 2.92 - Baidu Experience

Articles can only convey so much; some things are better seen in video. I found several great tutorials by experts on various video platforms.

Videos - Bilibili

Section titled “Videos - Bilibili”
  • Blender Beginner Basics Tutorial

(Didn’t really learn much). Just for VRChat; I don’t personally recommend it.

  • 【VRChat】Model Tutorial Collection: Uploading, Modding, Clothing, Weapon Toggles, Fixed Objects, Seats | Updated 2022.2.20

Created by Wonton. The first VRChat model upload video I watched. Very detailed.

  • 【VRChat & Unity Tutorial】Model Upload Basics & Common Effects Collection (Continuous Updates)

Created by RainyCloud. It’s from the SDK2 era but still good.

  • VRChat Model Interaction System New Dynamic Bone Tutorial (Physbone, Contact)

Created by Circes. Checked this out while writing the interaction system article; gave me a much clearer understanding.

  • 【VRChat Tutorial】VRC 3.0 Avatar Model Full Feature Breakdown - Unity Tutorial for Beginners and Intermediate Players (2.0 to 3.0)

Created by Mo Yan. This was the most helpful video (even though I found it last). Clearly explains SDK3.

  • 【VRChat】Easy Beginner Guide to Model Uploading & Dynamic/Collision Setup

Created by Yu Yan. Clear guide on converting and uploading MMD models.

  • 【VRC Toolbox 1.1】Major Update for VRChat Wardrobe Plugin!? Simple and Practical Unity Outfit Plugin

Created by Ru Meng. Release video for the VRC Toolbox.

  • [VRChat Modding] Understand the Principles of Changing Clothes in Unity in 10 Minutes

Created by ShineTrick. Clear explanation of changing clothes in Unity. Before this, I thought you had to use Blender if the clothes didn’t fit.

  • Becoming a VRChat Head-Swapping Expert Part 2! Advanced Head Swapping and Neck Interface Tutorial [LoliKami]

Created by LoliKamiGod. Main reference for the head-swapping article/video.

  • VRChat Model Modification using Blender (Version 2.82)

Created by Yami. Full recording of a model modification. Partial reference for the head-swapping article/video.
This guy likes listening to music on YouTube just like me.

  • 【VRChat Tutorial】Borrowing Special Effects from the Web + How to Make/Modify Particle Effects (Very confusingly explained)

  • VRChat Model Tutorial: From Head Swapping to Modifying Proportions and Fast Cloth. The ‘wild card’ of modeling tutorials.
